# DB(또는 파일)에 보관되는 데이터 목록

`MONGODB_URI` 설정 시 MongoDB에, 미설정 시 `data/` 폴더의 JSON 파일에 저장됩니다.

| 저장소 | 파일명 (파일 모드) | 보관 내용 |
|--------|-------------------|-----------|
| **회원** | `users.json` | 닉네임, 비밀번호 해시, 지갑 주소, 등급(level), 하트(points), 승인 여부, 자기소개(bio), 프로필 사진 URL, 닉네임 변경일, 상점 구매 내역 등 |
| **가이드 글** | `posts.json` | 가이드(게시판) 게시글·댓글 (제목, 내용, 작성자, 좋아요, 이미지 등) |
| **피드 글** | `feed.json` | 홈 피드 게시글·댓글 (내용, 작성자, 하트, 이미지 등) |
| **채팅** | `chat.json` | 폭풍 수다 채팅 메시지 (발신자, 내용, 시간 등) |
| **관리자 PIN** | `admin-pin.json` | 관리자 페이지 로그인용 PIN 해시 |
| **강제 탈퇴** | `force-withdraws.json` | 강제 탈퇴 처리된 지갑/회원 기록 |
| **상단 고정** | `pinned.json` | 채팅 상단 고정 메시지, 만료 시각 등 |
| **설정** | `settings.json` | 자동 승인 여부, 토네이도 뉴스 소스 등 관리자 설정 |

---

## 요약

- **회원/로그인**: `users` (닉네임, 비밀번호, 지갑, 등급, 하트, 프로필 등)
- **가이드**: `posts` (가이드 게시판 글·댓글)
- **홈 피드**: `feed` (피드 글·댓글)
- **채팅**: `chat` (채팅 메시지)
- **관리**: `admin-pin`, `force-withdraws`, `pinned`, `settings`

이 외에 **업로드된 프로필 사진 파일**은 `data/uploads/`(또는 설정한 업로드 경로)에 저장됩니다.
